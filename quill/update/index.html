download_archive_or_exit

If a version is provided, this function assumes that the tarball contains a version.txt file, and that the version should match the given expected version. Archive types tar.gz, tar.xz and zip are supported. If zip is provided, we assume that we’re downloading the zig compiler, and special-case logic is used to remove the top-level folder from the archive.
